COVID-19 dashboard and map built with Gatsby, React Leaflet, and Mapbox.

This dashboard includes data related to cases, deaths, and testing efforts. Information at a country level is displayed on cards that are activated on hover. Information at a global level (i.e. sum of all data) is displayed at the bottom of the page.

üìä Data

The data sources for this project are JHU CSSE COVID-19 Data, The New York Times, Worldometers, and Apple. These sources are updated in real time via the disease.sh endpoint as case data from around the world is reported.
